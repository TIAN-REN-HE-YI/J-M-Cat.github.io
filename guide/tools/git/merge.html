# 开发分支合并主分支

  1. 先切换到开发分支
git checkout dev
  1. 拉取分支
git pull
  1. 切换到主分支
git checkout master
  1. 合并代码到主分支
git merge dev
  1. 提交代码
git push

# 主分支合并到开发分支

  1. 先切换到主分支
git checkout master
  1. 拉取代码
git pull
  1. 切换到开发分支
git checkout dev
  1. 主分支代码合并到开发分支
git merge master
  1. 提交代码
git push
