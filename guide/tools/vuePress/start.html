    # 安装步骤

    如果你只是想随便用下 VuePress，你可以在全局安装它：
    npm install -g vuepress

    创建项目文件夹：
    mkdir vuepress-demo

    进入项目文件夹：
    cd vuepress-demo

    初始化项目：
    npm init

    然后你就可以开始编写文档了：
    npm run docs:dev

    要生成静态资源，请运行：
    npm run docs:build
